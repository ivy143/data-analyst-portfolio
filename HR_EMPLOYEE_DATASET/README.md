# HR Employee Analytics (SQL Project)

## Project Overview

This project analyzes HR employee data to understand workforce trends, employee attrition, and salary distribution.
The goal is to use SQL to extract insights that can help organizations improve employee retention and workforce management.

## Objective

* Analyze employee attrition patterns
* Evaluate salary distribution across departments
* Understand workforce demographics
* Identify high-performing employees

## Dataset

The dataset contains employee-related information such as:

* Employee ID
* Department
* Gender
* Salary
* Years of Experience
* Performance Rating
* Attrition (Yes/No)

## Tools & Technologies

* SQL
* Relational Database (MySQL / PostgreSQL / SQL Server)

## Key Analysis Performed

### 1. Attrition Rate Analysis

* Calculated percentage of employees leaving the company
* Identified patterns in employee turnover

### 2. Department-wise Salary Analysis

* Calculated average salary across departments
* Compared compensation levels

### 3. Gender Distribution

* Analyzed male vs female employee ratio
* Studied workforce diversity

### 4. Experience vs Salary

* Explored relationship between experience and salary
* Identified trends in compensation growth

### 5. High Performer Identification

* Identified employees with high performance ratings
* Analyzed their characteristics

### 6. Advanced SQL Techniques

* Used Common Table Expressions (CTE)
* Applied subqueries for complex analysis

## SQL Concepts Used

* Aggregations (AVG, COUNT)
* GROUP BY
* Joins
* Subqueries
* Common Table Expressions (CTE)

## Key Insights

* Attrition rate highlights areas where employee retention needs improvement.
* Salary distribution varies significantly across departments.
* Experience generally correlates with higher salary.
* High-performing employees can be identified using performance metrics.

## Conclusion

This project demonstrates how SQL can be used for HR analytics and workforce insights. It highlights the importance of data-driven decision-making in managing employees and improving organizational performance.
