# IPL Data Analysis (EDA)

## Project Overview

This project performs exploratory data analysis on IPL (Indian Premier League) cricket data to uncover patterns related to teams, players, and match outcomes.
The goal is to analyze historical match data and extract insights using Python data analysis libraries.

## Objective

* Analyze team and player performance
* Identify winning patterns and trends
* Explore factors influencing match outcomes

## Dataset

The dataset contains IPL match data with fields such as:

* Match ID
* Season
* Teams
* Venue
* Toss Winner
* Match Winner
* Player of the Match
* Runs / Wickets

## Tools & Technologies

* Python
* Pandas
* Matplotlib
* Seaborn

## Project Workflow

### 1. Data Cleaning

* Handled missing values
* Standardized team names and formats
* Removed inconsistencies in dataset

### 2. Exploratory Data Analysis

* Analyzed match outcomes by team
* Studied toss impact on match results
* Evaluated player performance trends
* Compared team win percentages

### 3. Data Visualization

* Team performance charts
* Win distribution graphs
* Player performance analysis
* Match outcome trends
