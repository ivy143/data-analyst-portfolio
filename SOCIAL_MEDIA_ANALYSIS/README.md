# Social Media Analytics Dashboard (Power BI)

## Project Overview

This project analyzes social media performance data from platforms like Instagram, YouTube, and Twitter.
The goal is to track engagement, audience growth, and content performance using an interactive Power BI dashboard.

## Objective

* Monitor key social media KPIs
* Analyze engagement trends over time
* Identify high-performing content
* Understand audience growth and behavior

## Dataset

The dataset includes social media analytics data such as:

* Post Date
* Platform (Instagram / YouTube / Twitter)
* Followers
* Likes
* Comments
* Shares
* Reach
* Impressions
* Content Type (Reels / Posts / Videos)

## Tools & Technologies

* Power BI
* Data Visualization
* Data Analysis

## Dashboard Structure

### 1. Overview Page

Provides a high-level summary of performance:

* Total Followers
* Engagement Rate
* Total Posts
* Total Likes & Comments

### 2. Engagement Analysis

Analyzes how users interact with content:

* Engagement trend over time
* Best posting day and time
* Top performing posts
* Content type performance (Reels / Posts / Videos)

### 3. Audience Insights

Focuses on audience growth and reach:

* Follower growth trend
* Audience demographics
* Reach vs Impressions analysis

## Key Insights

* Engagement varies significantly based on posting time and content type.
* Certain content formats (like short videos) tend to generate higher interaction.
* Follower growth trends help measure long-term performance.
* Reach and impressions provide insight into content visibility.

## Conclusion

This dashboard helps track and improve social media performance by providing clear insights into engagement, audience behavior, and content effectiveness. It supports data-driven decisions for optimizing content strategy.
