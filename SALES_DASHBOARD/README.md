# Sales Dashboard (Excel)

## Project Overview

This project transforms raw sales data into an interactive Excel dashboard to analyze business performance.
The goal is to clean data, create pivot tables, and build a dashboard that provides insights into sales trends, regional performance, and product analysis.

## Objective

* Convert raw sales data into meaningful insights
* Analyze sales performance across different dimensions
* Build an interactive dashboard using Excel

## Dataset

The dataset contains sales-related information such as:

* Order Date
* Region
* Product Name
* Category
* Sales
* Profit
* Order Quantity

## Tools & Technologies

* Microsoft Excel
* Pivot Tables
* Data Cleaning
* Data Visualization

## Project Workflow

### 1. Data Cleaning

* Removed missing or incorrect values
* Standardized data format
* Prepared dataset for analysis

### 2. Data Transformation

* Created pivot tables
* Aggregated data by category, region, and time

### 3. Dashboard Creation

Built an interactive dashboard including:

* Monthly Sales Trend
* Region-wise Sales Analysis
* Category-wise Revenue
* Top 10 Products by Sales
* KPI Cards:

  * Total Revenue
  * Total Profit
  * Total Orders

## Key Insights

* Sales trends show variations across different months, indicating seasonal patterns.
* Certain regions contribute significantly more to total revenue.
* A small number of products generate the majority of sales.
* Category-wise analysis helps identify high-performing product segments.

## Conclusion

This project demonstrates how Excel can be used as a powerful tool for data analysis and dashboard creation. It highlights the importance of data cleaning, aggregation, and visualization in business decision-making.
