# Retail Sales Analysis

## 📌 Project Overview
This project focuses on analyzing retail sales data to extract meaningful business insights. The goal is to understand sales performance, customer behavior, and product trends to support data-driven decision-making.

---

## 🎯 Problem Statement
Retail businesses generate large amounts of sales data, but without proper analysis, it is difficult to identify key trends. This project aims to analyze sales data to uncover insights such as top-performing products, revenue trends, and customer contribution.

---

## 📂 Dataset Information
- Dataset: Superstore / Retail Sales Dataset
- Features include:
  - Order ID
  - Customer Name
  - Product Name
  - Category
  - Sales
  - Profit
  - Region
  - Order Date

---

## 🧹 Data Cleaning
- Checked for missing values
- Removed duplicates
- Standardized column formats
- Converted date column to proper format

---

## 📊 Key Analysis Performed

### 🔝 Top Selling Products
- Identified products generating highest revenue

### 📅 Monthly Revenue Trends
- Analyzed sales over time to identify seasonality and trends

### 💰 Profit by Category
- Compared profitability across different product categories

### 👥 Top 10 Customers
- Identified highest revenue-generating customers

### 🌍 Region-wise Performance
- Analyzed which regions contribute most to sales and profit

### 🏆 Ranking using Window Functions
- Used SQL window functions like `RANK()` to find top-performing products

---

## 🧠 SQL Concepts Used
- Aggregations (SUM, AVG)
- GROUP BY
- JOINS
- ORDER BY
- Window Functions (RANK, DENSE_RANK)

---

## 📈 Insights
- Identified best-selling products and categories
- Found top customers contributing major revenue
- Discovered high-performing regions
- Observed monthly trends in revenue

---

## 💡 Business Recommendations
- Focus marketing on top-performing products
- Improve sales strategies in low-performing regions
- Retain high-value customers with special offers
- Optimize product categories with low profit

---

## 🛠 Tools & Technologies
- SQL (MySQL / PostgreSQL)
- Excel (for initial data viewing)

---

## 📌 Conclusion
This project demonstrates how SQL can be used to analyze business data and generate actionable insights that help improve decision-making in retail businesses.
