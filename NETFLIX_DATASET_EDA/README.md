# Netflix Dataset Exploratory Data Analysis (EDA)

## Project Overview

This project performs exploratory data analysis on the Netflix titles dataset to understand trends in movies and TV shows available on the platform.
The goal is to explore content distribution, genres, release patterns, and platform growth using Python data analysis libraries.

## Dataset

The dataset contains information about Netflix movies and TV shows.

Typical fields include:

* title
* type (Movie / TV Show)
* director
* cast
* country
* release_year
* rating
* duration
* listed_in (genre)
* date_added

## Tools & Libraries Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Project Workflow

### 1. Data Cleaning

* Handled missing values
* Standardized column formats
* Converted date fields for analysis
* Removed duplicate records

### 2. Exploratory Data Analysis

* Analyzed distribution of Movies vs TV Shows
* Studied content release trends over the years
* Identified the most common genres
* Explored country-wise content production

### 3. Data Visualization

Visualizations were created using Matplotlib and Seaborn to understand patterns and trends in the dataset.

## Key Insights

* Movies make up the majority of content available on Netflix compared to TV shows.
* Netflix significantly increased content additions after 2015.
* Drama and Comedy appear among the most common genres.
* The United States contributes the largest portion of content.

## Conclusion

The analysis provides insights into Netflix’s content distribution and growth trends. Understanding these patterns can help evaluate the platform's content strategy and audience preferences.
