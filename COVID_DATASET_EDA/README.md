# COVID-19 Data Analysis (EDA)

## Project Overview

This project performs exploratory data analysis on COVID-19 datasets to understand the spread, impact, and trends of the pandemic across different regions.
The goal is to analyze case trends, recoveries, and deaths using Python data analysis libraries.

## Objective

* Analyze COVID-19 case trends over time
* Compare impact across different countries/regions
* Understand recovery and death patterns
* Visualize pandemic trends using graphs

## Dataset

The dataset contains COVID-19 related data such as:

* Date
* Country / Region
* Confirmed Cases
* Recovered Cases
* Deaths
* Active Cases

## Tools & Technologies

* Python
* Pandas
* Matplotlib
* Seaborn

## Project Workflow

### 1. Data Cleaning

* Handled missing values
* Converted date columns into proper format
* Standardized country/region names
* Removed duplicate records

### 2. Exploratory Data Analysis

* Analyzed total cases over time
* Compared confirmed, recovered, and death cases
* Studied country-wise impact
* Identified growth patterns

### 3. Data Visualization

* Line charts for trend analysis
* Bar charts for country comparison
* Distribution plots for case analysis
* Time series visualization

## Key Insights

* COVID-19 cases showed exponential growth during initial phases.
* Certain countries experienced significantly higher case counts.
* Recovery rates improved over time in many regions.
* Death rates varied based on region and time period.

## Conclusion

This project demonstrates how data analysis can be used to understand global events and trends. By analyzing COVID-19 data, the project provides insights into the spread and impact of the pandemic across different regions.
