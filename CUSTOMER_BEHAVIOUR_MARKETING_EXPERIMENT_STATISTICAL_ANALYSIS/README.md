# Customer Behavior & Marketing Analytics

## Project Overview

This project analyzes customer behavior and marketing performance using statistical analysis, probability models, and A/B testing.
The objective is to understand spending patterns, evaluate business risks using probability analysis, and compare marketing campaign performance using statistical hypothesis testing.

## Objective

* Analyze customer spending behavior using statistical techniques.
* Estimate business risks using probability models.
* Evaluate marketing campaign performance using A/B testing.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Statistical Analysis

## Analysis Modules

### Module 1 – Customer Behavior Statistical Analysis

This module explores customer spending patterns using statistical methods.

Analysis includes:

* Distribution analysis
* Normal vs skewed distribution detection
* Correlation analysis between variables
* Outlier detection
* Z-score calculation

Statistical concepts used:

* Mean, Median, Mode
* Variance and standard deviation
* Correlation analysis
* Z-score
* Distribution analysis

### Module 2 – Probability & Business Risk Analysis

This module estimates potential business risks using probability models.

Example use cases:

* Loan default probability
* Customer churn probability
* Sales prediction probability

Techniques used:

* Probability rules
* Conditional probability
* Expected value calculation
* Basic regression introduction

Concepts covered:

* Probability fundamentals
* Conditional probability
* Expected value
* Risk estimation

### Module 3 – A/B Testing Analysis

This module compares two marketing campaigns to determine which performs better.

Example scenario:

* Website Version A vs Version B
* Conversion rate comparison

Analysis steps:

* Mean comparison
* Hypothesis testing
* p-value calculation
* Confidence interval estimation
* Statistical significance evaluation

Statistical concepts used:

* Mean and standard deviation
* Hypothesis testing
* p-value
* t-test
* Confidence intervals

## Key Insights

* Customer spending patterns often show skewed distributions rather than normal distributions.
* Probability models can estimate potential business risks such as churn or loan default.
* A/B testing helps businesses identify the most effective marketing strategy using statistical evidence.

## Conclusion

This project demonstrates how statistical techniques and probability models can be applied to real-world business problems. By combining customer behavior analysis, risk modeling, and A/B testing, the project shows how data-driven decisions can improve marketing strategies and business outcomes.
