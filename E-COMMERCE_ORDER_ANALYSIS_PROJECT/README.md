# E-commerce Order Analysis (SQL + Power BI)

## Project Overview

This project analyzes e-commerce order data to understand sales performance, customer behavior, and key business metrics.
SQL is used for data analysis, and Power BI is used to create an interactive dashboard for visualization and storytelling.

## Objective

* Analyze sales trends and customer purchasing behavior
* Identify repeat and new customers
* Evaluate product category performance
* Build a dashboard to track key business KPIs

## Dataset

The dataset contains e-commerce order data with fields such as:

* Order ID
* Order Date
* Customer ID
* Product Category
* Sales Amount
* Quantity

## Tools & Technologies

* SQL
* Power BI
* Data Modeling
* Data Visualization

## SQL Analysis

### 1. Monthly Sales Analysis

* Calculated total sales for each month
* Identified trends and seasonality

### 2. Customer Retention Analysis

* Measured how many customers returned for repeat purchases
* Identified retention patterns

### 3. Repeat vs New Customers

* Segmented customers into new and returning
* Compared their contribution to total sales

### 4. Top Categories

* Identified best-performing product categories
* Ranked categories based on revenue

### 5. Average Order Value (AOV)

* Calculated average revenue per order
* Used to evaluate customer spending behavior

## Power BI Dashboard

The dashboard includes:

* KPI Cards (Total Sales, Total Orders, AOV)
* Monthly Sales Trend Chart
* Category-wise Sales (Pie Chart)
* Customer Retention Trend

## Key Insights

* A significant portion of revenue comes from repeat customers.
* Certain product categories consistently outperform others in sales.
* Monthly sales trends reveal peak business periods.
* Average Order Value helps understand customer spending patterns.

## Learning Outcomes

* SQL-based business analysis
* Understanding of key business KPIs
* Basic data modeling for dashboards
* Data storytelling using Power BI

## Conclusion

This project demonstrates how SQL and Power BI can be combined to analyze e-commerce data and present actionable insights. It highlights the importance of customer retention, category performance, and KPI tracking in business decision-making.
